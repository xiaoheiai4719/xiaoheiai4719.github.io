第一篇文章 使用hexo+git 搭建一个技术博客

  1. 注册github。
  2. 新建仓库。
  3. 本地安装 npm install hexo-cli -g

  4. hexo init 你的用户名.github.io // 尽量和Github仓库使用同一个名称。这个地方 需要的node的版本 。我升级了一下 我本地的node到最新，结果他不支持。所以 这个地方一定选择一个合适的node版本。

    deploy:
    type:git
    repo:仓库地址
    branch:master
    `

  5. hexo deploy 即可部署到线上
